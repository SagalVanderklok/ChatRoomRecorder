About the app


ChatRoomRecorder is an open source application for recording Chaturbate, BongaCams, Stripchat, Flirt4Free, CamSoda and Cam4 shows with no limitations!

How to use

  1. Download and install .NET Desktop Runtime (x64 or x86) and WebView2 Runtime (x64 or x86)
  2. Download and install Streamlink (x64 or x86)
  3. Download and unpack ChatRoomRecorder (x64 or x86)
  4. Start ChatRoomRecorder
  5. Open "Settings" tab and specify Streamlink and ffmpeg binaries locations
  6. Open "Web browser" tab, visit desired web sites and pass bot protection
  7. Open "Chat rooms" tab, add chat room links and start recording
